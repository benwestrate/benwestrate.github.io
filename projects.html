LumberJack

LumberJack is a modern client side-logging framework. The problems addressed with LumberJack are:

  1. Clear definition of where each log is coming from.
    • Title of the component where the log comes from
    • Consistant color of the log comming from the component
    • Full stack trace included in each log
  2. Full server side logging of configured log level.
    • Including the stack trace

This is an open source project still in process.
